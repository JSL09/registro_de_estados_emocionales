Descripción del Programa
Este programa es una herramienta de evaluación de estado emocional que utiliza un algoritmo para registrar y analizar síntomas. Basado en la entrada del usuario, el programa determina si está en un estado de hipomanía, manía, depresión o normalidad. Desarrollado en Python con React, el objetivo principal es servir como una herramienta útil para la salud mental, proporcionando a los usuarios una manera de autoevaluarse y entender mejor su estado emocional.
Características:
    • Registro de síntomas personalizados.
    • Evaluación automatizada del estado emocional.
    • Basado en un algoritmo lógico de clasificación.
